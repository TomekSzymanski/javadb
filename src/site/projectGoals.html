Only release 1 scope is implemented.

Release 1

  • Embedded mode
  • Database session
  • Very basic DDL support: only CREATE TABLE and DROP TABLE,
  • Very basic DML: INSERT, DELETE without WHERE, SELECT from one table without WHERE condition
  • Simple data types (for columns)
  • System dictionary
  • Data written to disk as pages. Support actual record layout as byte array
  • Buffer cache

Release 2

  • JDBC driver, client-server mode (connectivity over TCP)
  • SELECT with joins, 3 join methods supported (relational query processor)
  • Indices, plain B-tree

Release 3

  • UPDATE
  • Sorting (ORDER BY) : also with data sizes exceeding RAM (simulate with smaller Xmx): external sorts
  • Query optimiser, statistics

Release 4

  • Any transaction support, ACID at best
  • Multi-user, concurrency

Scope backlog

  • Basic authentication and authorization
  • Referential integrity (foreign keys)
  • Crash-recovery
  • Backup
  • Schemas (users)
  • UNION, GROUP BY
  • ALTER TABLE
  • Strong performance automated and regression testing
  • Encryption of network connections (client-server)
  • Stored procedures (in java) and java functions
  • Binary datatype
  • Triggers
  • Views
  • Temporary tables
  • Other indices (bitmap)
  • Partitioning
  • Role based access control
  • windowing functions
